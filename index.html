      How do I get a event ID?

      1. Click to “Share” the Campfire event; this should generate a link like https://cmpf.re/my58qK.
      2. Paste this into a browser; the browser should navigate to a URL like https://campfire.nianticlabs.com/discover/meetup/60e66913-7a36-4835-96fb-47fe7c5d3192. Copy that URL into the box above (one event per line). You can paste the full URL — the event ID is parsed automatically.
